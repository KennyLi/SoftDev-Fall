Grid System

Starting from full screen, try resizing the window so that it gets smaller and smaller. You will notice that the two paragraphs will shift from a 1:1 ratio to a 2:1 ratio to a 1:2 ratio. The left cell is lg-6/md-9/sm-3 and the right cell is lg-6/md-3/sm-9. When the window is large (>992px), both paragraphs take up 6 out of 12 columns which is 1:1. When the window is medium (>768px), the left cell takes up twice as much as the right cell which is 2:1. When the window is small (>576px), the left cell takes up half as much as the right cell which is 1:2.

Buttons

Here are bootstrap's predefined button styles and their semantic purpose.

Coloring

Changes the button's color.

<button type="button" class="btn btn-primary"></button> #To get the other buttons, just replace "primary" with the label associated with each button above.
Sizing

Changes the button's size. Block will change the width of the button to fill the entire container.

<button type="button" class="btn btn-sm"></button> #To get the other buttons, just replace "sm" with the label associated with each button above.
Others

Outline must be used with one of the colored buttons to remove the background.

<button type="button" class="btn" disabled></button> <button type="button" class="btn btn-outline-primary"></button>
